Projeto Java: Treinando
Este repositório reúne exercícios e práticas desenvolvidos durante os estudos de Java pela Alura, com foco em fundamentos da linguagem como condicionais, laços de repetição, leitura de dados e estrutura básica de programas em Java. Pelos arquivos listados, o projeto está organizado em classes separadas por assunto, como Condicional.java, Leitura.java, Loop.java, OutroLoop.java, Main.java e exercicio.java, o que indica uma proposta de treino progressivo dos principais conceitos iniciais da linguagem.

Objetivo
O objetivo deste projeto é praticar a base da programação com Java por meio de exemplos simples e diretos, reforçando a lógica e o uso da sintaxe da linguagem. Esse tipo de organização por arquivos temáticos é comum em trilhas introdutórias e ajuda a consolidar estruturas como if/else, for, while, variáveis e entrada de dados com Scanner.

Conteúdos praticados
Condicionais: decisões com if, else if e else, normalmente usadas para validar regras e controlar o fluxo do programa.

Leitura de dados: captura de informações digitadas pelo usuário, geralmente com Scanner, para tornar os exercícios interativos.

Loops: repetição com estruturas como for e while, fundamentais para automatizar tarefas e percorrer sequências.

Estrutura básica em Java: criação de classes, método main e execução de pequenos programas no console.

Estrutura do projeto
Pelos arquivos informados, o repositório contém exemplos separados por tema, o que facilita a leitura e o aprendizado incremental. A presença de classes como Condicional.java, Leitura.java, Loop.java e OutroLoop.java sugere um projeto voltado para prática guiada e reforço dos conceitos iniciais de Java.

Aprendizados
Esse projeto ajuda a desenvolver a lógica de programação e a familiaridade com a sintaxe do Java em pequenos exercícios de console. Ele também funciona como base para projetos futuros mais estruturados, inclusive aqueles que usam orientação a objetos, coleções e tratamento de exceções.

Como executar
Clone o repositório.

Abra o projeto em uma IDE compatível com Java.

Execute a classe desejada conforme o tema que quer testar, como Main, Condicional ou Loop.

Observe a saída no console e, nos exercícios com leitura, informe os valores solicitados.

Observação
Este repositório representa uma etapa de aprendizado e prática, reunindo exercícios das aulas da Alura em Java. A proposta é servir como material de treino e evolução, acompanhando o desenvolvimento dos fundamentos da linguagem.
